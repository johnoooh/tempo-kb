# Computing Variant Allele Frequency (VAF) from Tempo MAF Files

> **Quick answer:** Tumor VAF is calculated as `t_alt_count / t_depth` (or equivalently `t_alt_count / (t_alt_count + t_ref_count)`). Normal VAF uses the corresponding normal columns and should be near zero for true somatic mutations.

## What Is VAF

Variant Allele Frequency (VAF) is the fraction of sequencing reads supporting the alternate (mutant) allele at a given position. VAF reflects the proportion of cells carrying the mutation, modulated by copy number and tumor purity. It is a fundamental metric for assessing mutation confidence, tumor heterogeneity, and clonality.

## VAF Formulas

### Tumor VAF

```
tumor_vaf = t_alt_count / t_depth
```

This is equivalent to:

```
tumor_vaf = t_alt_count / (t_alt_count + t_ref_count)
```

### Normal VAF

```
normal_vaf = n_alt_count / n_depth
```

For true somatic mutations, normal VAF should be approximately 0. Elevated normal VAF may indicate germline variants, sequencing artifacts, or contamination.

## Caller Counts vs Raw Counts

The Tempo MAF contains two sets of allele count columns:

| Column Set | Source | Use Case |
|------------|--------|----------|
| `t_alt_count`, `t_ref_count`, `t_depth` | Variant caller (Mutect2/Strelka2) | Standard VAF calculation |
| `t_alt_count_raw`, `t_ref_count_raw`, `t_depth_raw` | GetBaseCountsMultiSample (GBCMS) | Independent count verification |

The caller counts come directly from the variant caller that detected the mutation. The raw counts from GBCMS are generated by re-counting reads at each variant position independently of the caller. Use the caller counts (`t_alt_count`, `t_depth`) for standard VAF calculations. The raw counts are useful for cross-validation or when filling back mutations across samples in a cohort.

## Coverage Thresholds

Tempo applies default filtering thresholds during variant calling:

- **Tumor depth**: `t_depth >= 20`
- **Normal depth**: `n_depth >= 10`

Variants below these thresholds are filtered out in the final MAF. If you are working with the unfiltered MAF, apply these thresholds manually to ensure reliable VAF estimates.

## File Locations

```
outDir/somatic/{idTumor}__{idNormal}/combined_mutations/{idTumor}__{idNormal}.somatic.final.maf
```

## Example: Python

```python
import pandas as pd

maf_path = "outDir/somatic/TUMOR__NORMAL/combined_mutations/TUMOR__NORMAL.somatic.final.maf"
maf = pd.read_csv(maf_path, sep='\t', comment='#', low_memory=False)

# Calculate tumor and normal VAF
maf['tumor_vaf'] = maf['t_alt_count'] / maf['t_depth']
maf['normal_vaf'] = maf['n_alt_count'] / maf['n_depth']

# Summary statistics
print(maf['tumor_vaf'].describe())
print(f"Median tumor VAF: {maf['tumor_vaf'].median():.3f}")
print(f"Median normal VAF: {maf['normal_vaf'].median():.4f}")
```

### Filtering by VAF

```python
# High-confidence somatic variants
confident = maf[
    (maf['tumor_vaf'] >= 0.05) &   # at least 5% tumor VAF
    (maf['normal_vaf'] <= 0.01) &   # near-zero normal VAF
    (maf['t_depth'] >= 20) &        # adequate tumor coverage
    (maf['n_depth'] >= 10)          # adequate normal coverage
].copy()
print(f"High-confidence variants: {len(confident)}")
```

### VAF Distribution Plot

```python
import matplotlib.pyplot as plt

fig, axes = plt.subplots(1, 2, figsize=(12, 4))

axes[0].hist(maf['tumor_vaf'].dropna(), bins=50, edgecolor='black')
axes[0].set_xlabel('Tumor VAF')
axes[0].set_ylabel('Count')
axes[0].set_title('Tumor VAF Distribution')

axes[1].hist(maf['normal_vaf'].dropna(), bins=50, edgecolor='black')
axes[1].set_xlabel('Normal VAF')
axes[1].set_ylabel('Count')
axes[1].set_title('Normal VAF Distribution')

plt.tight_layout()
plt.savefig('vaf_distributions.png', dpi=150)
```

## Interpreting VAF

- **VAF near 0.5**: Heterozygous mutation in a diploid region at high purity
- **VAF near 1.0**: Homozygous mutation or loss of heterozygosity (LOH)
- **VAF < 0.1**: Subclonal mutation or low-purity sample
- **Normal VAF > 0.05**: Suspect germline variant or contamination

VAF alone does not directly indicate clonality. To determine whether a mutation is clonal or subclonal, use the cancer cell fraction (CCF), which adjusts VAF for tumor purity and local copy number.
